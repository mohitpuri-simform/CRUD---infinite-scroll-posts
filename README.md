# Product Listing Application

## Overview

This project is a **Post Listing Application** built using the **MVC (Model-View-Controller) design pattern**. It provides a structured way to display a list of posts with infinite scroll.

## Features

- **MVC Architecture** for better code organization.
- **Infinte scrolling** to navigate through multiple posts.
- **HTML & CSS** for UI design.
- **Parcel** as the build tool for bundling and performance optimization.
